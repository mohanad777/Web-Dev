HTML & CSS Basics Summary

HTML Summary

HTML (HyperText Markup Language) is the heart of every webpage. It defines the structure of the webpage.

  • HTML elements are typically made up of an opening tag
  • HTML elements can also receive attributes that allow you to add extra behavior
  • Elements for presenting and describing your page content
    (e.g. h1, p, ul, a, img etc.)
  • Elements for describing your overall page and for linking to other required resources
    (e.g. title, meta, link, style)
